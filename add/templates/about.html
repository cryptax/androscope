Motivation

"Please send SHA256 of sample using a native packer"

Although there are several such samples, you'll probably waste hours trying to find such a sample again. There are so many Android (several millions), it is difficult to spot one in particular. This is what Androscope is for.

Androscope keeps track of malicious features for each sample, and offers the possibility to quickly search among them. For example, you can easily search a malware which implement Anti-Frida techniques, or a malware which turns off the ringer etc.
Each new entry in androscope consists in a list of implemented features. The list is extensible by the use of several keyword fields.
Note that Androscope does not store / keep any malware: only their hash and list of features.
