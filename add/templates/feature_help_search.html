Help - Searching for malware

Is it possible to search malware by name?

Yes. Select the Main category, then specify malware name in the Malware Name field.

How to search malware by SHA256?

Select the Main category, then specify the desired SHA256 hash.

Combining features

If you select multiple features, an AND operator is automatically applied between them.
Currently, you cannot apply an OR or specify complex operations.

Example:
To search for malware which intercept incoming SMS, click on the toggle button "SMS", then select "Intercept incoming SMS", finally press button "Finished - Submit" to start the search.
If you select "keylogger" (main category) and "some (or all) malicious parts are implemented in a native library" (native), Androscope searches for a malware which implements a keylogger and uses a native library.

What does the "anything else" feature mean in the Main category?

If there is no specific feature for what you want to search, this is the category to use.
